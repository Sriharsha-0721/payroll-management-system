# 📊 Payroll Management System

## 🚀 Overview
A payroll system developed to manage employee data, salary processing, and report generation.

## 🛠️ Features
- Employee Management  
- Salary Calculation  
- Payslip Generation  
- Monthly Reports  
- Login & Dashboard  

## 🧠 Technologies Used
- Progress OpenEdge ABL  
- AppBuilder  
- CSV Data Handling  

## 💡 Description
This project demonstrates real-world payroll processing with modular design and structured data handling.

## 📌 Note
This project is built using OpenEdge ABL and is not a web-based application.
